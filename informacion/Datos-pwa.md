# PWA GAMIFICACIÓN ONLINEMENTE

## Pantalla de login 

- Imagen corporativa del proyecto.
- Inputs para usuario y contraseña.
- Botón de iniciar sesión.

# Rol Usuario

- Debe poder configurar su perfil
- Debe poder ver su ranking en la página de ranking
- Debe poder ver las monedas que tiene
- Debe poder ver las insignias que tiene
- Debe poder ver las misiones que tiene
- Debe poder ver los logros que tiene
- Debe poder ver los retos que tiene
- Debe poder ver los niveles que tiene
- Debe poder ver los puntos que tiene
- Debe poder ver los premios que tiene
- Debe tener un calendario diario en el que poder añadir el ejercicio diario, tanto el tipo de ejercicio como la duracion, para que se guarden en la base de datos
- Debe poder ver un historial de los ejercicios diarios que ha realizado
- Debe poder tener una casilla donde introducir los pasos diarios que ha realizado
- Debe poder ver un historial de los pasos diarios que ha realizado
- Debe poder ver un dashboard con toda la información de su actividad diaria

# Rol Admin

- Debe tener todas las funciones del rol usuario más las siguientes.
- Crear o eliminar usuarios
- Crear o eliminar insignias
- Crear o eliminar misiones
- Crear o eliminar retos
- Crear o eliminar niveles
- Crear o eliminar puntos
- Crear o eliminar premios
- Crear o eliminar tipo de ejercicio
- Crear o eliminar premios

- Debe de aparecer cuantos puntos te faltan para conseguir ese premio o un check si está conseguido

# Página de misiones

- Debe de aparecer cuantos puntos te faltan para conseguir ese reto o un check si está conseguido

# Página de niveles
- Debe de ver un listado de todos los niveles disponible

# Página de puntos
- Debe de ver un listado de todos los puntos disponible
- Debe de aparecer cuantos puntos te faltan para conseguir ese reto o un check si está conseguido

# Calendario
- Debes poder añadir actividades nuevas cada día con los siguientes campos:
1. Tipo de ejercicio
2. Duración
3. Adjuntar una foto que debera almacenarse en el servidor y aparecer en la ficha del ejercicio de cada persona
4. Elegir la intensidad entre: Suave, moderado, titán
- Además cada día debe tener los puntos de los pasos diarios. Debe aparecer un input donde se introduzca los pasos diarios y se guarde en la base de datos, si no se introduce nada no cuenta como 0 puntos.
- En el calendario debe aparecer un check en los días que se han realizado actividades o se han introducido pasos diarios.

# Sistema de puntos
- Cada paso diario valdra 0,0001 puntos
- Cada hora de entrenamiento valdra 100 puntos

# Premios
 - Se añadirán desde el panel de administración
 - Debe tener los siguientes campos:
 1. Nombre del premio
 2. Descripción
 3. Imagen
 4. Puntos necesarios para conseguirlo
 5. Fecha de caducidad
